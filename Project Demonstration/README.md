# Project Demonstration

## Demo Steps

1. Launch the Flask application.
2. Open the application in a web browser.
3. Enter soil and environmental values.
4. Click the Predict button.
5. The system displays the recommended crop.

## Expected Output

The application predicts the most suitable crop based on the given input values.

## Conclusion

The demonstration shows that the system successfully recommends crops using a machine learning model, helping farmers make better agricultural decisions.
