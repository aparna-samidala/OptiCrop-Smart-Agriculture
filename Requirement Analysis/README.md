# Requirement Analysis

## Functional Requirements

- User should be able to enter soil and environmental parameters.
- System should predict the most suitable crop.
- Display prediction results instantly.

## Non-Functional Requirements

- User-friendly interface.
- Fast prediction response.
- Accurate machine learning model.
- Reliable and scalable application.

## Software Requirements

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML, CSS

## Hardware Requirements

- Intel i3 or higher processor
- 4 GB RAM or above
- 500 MB free disk space
